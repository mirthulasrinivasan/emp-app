
# Employee Management App

This is a full-stack web application with:

- 🧠 **Django** (backend)
- 🔥 **Ember.js** (frontend)
- 🐘 **PostgreSQL** (database)

## Features
- Secure login & logout
- Add, view, edit, delete employee details
- JSON-based API communication between frontend and backend
- Protected views for authenticated users
